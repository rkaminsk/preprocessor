Detailed Description

An atom base used to store derivable atoms and associated state.

The base tracks the generation of atoms for semi-naive evaluation, and the state of atoms.

An atom base can also stores unknown atoms. For such atoms it is not yet know whether there will be a rule deriving them. The only purpose is to store them here is to associated them with a unique id.
